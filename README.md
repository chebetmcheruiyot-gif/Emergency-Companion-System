🚨 Emergency Companion System
A real-time emergency response application designed to assist users during emergency situations by providing instant guidance and alerting trusted contacts.

📌 About The Project
The Emergency Companion System was built to address the challenge of delayed emergency response, particularly in areas where calling for help may not always be possible. The system provides users with a quick, reliable way to alert contacts and access emergency guidance from any device.

✨ Features

🆘 One-click emergency alert to pre-saved contacts
📍 Real-time guidance based on emergency type (medical, fire, security, etc.)
📱 Responsive interface accessible on both desktop and mobile
👥 Contact management — add and manage trusted emergency contacts
📋 Emergency checklist and instructions per situation type


🛠️ Built With

HTML — structure and layout
CSS — styling and responsive design
JavaScript — interactivity and real-time functionality
Python / Flask — back-end logic and alert handling


🚀 How To Run

Clone the repository:

bash   git clone https://github.com/chebetmcheruiyot-gif/emergency-companion-system.git
   cd emergency-companion-system

Install dependencies:

bash   pip install -r requirements.txt

Run the application:

bash   python app.py

Open your browser and go to:

   http://localhost:5000

💡 Use Cases

Medical emergencies at home or in public
Security threats or unsafe situations
Accidents requiring immediate assistance
Campus or institutional emergency response
